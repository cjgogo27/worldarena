AutoGeo: Autonomous Subagent Scheduling for Geolocalization

Overview
- Lightweight, pluggable prototype implementing a master-subagent orchestration for multi-image geolocalization tasks inspired by MultiLoc.
- Three subagents with distinct tool-theme collaborate to deduce city-level location from a set of images/scenes.

Usage
- Run the built-in demo: node autogeo/demo.js
- Or wire it into the existing backend as /autogeo/run to test via REST.

Notes
- This is a prototype with mocked tools and synthetic reasoning for demonstration purposes.
- The goal is to provide a structured, extensible codebase that mirrors the architecture discussed in the document you shared.
